### Accession
PXD024739

### Title
Milk-derived extracellular vesicles

### Description
Oral administration of bovine milk-derived extracellular vesicles induces senescence in the primary tumor but accelerates cancer metastasis

### Sample Protocol
EVs were isolated by differntial centrifugation coupled with ultracentrifugation

### Data Protocol
Peak lists were extracted from raw mass spectrometry into the Mascot Generic File Format (MGF) using MsConvert with peak picking. The MGF files were then searched using X! Tandem.
