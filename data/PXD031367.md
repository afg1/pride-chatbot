### Title
Phosphoproteomic analysis of STK25 knockout iPS derived cardiomyocytes

### Description
Analysis of phosphoproteomic changes with knockout of STK25 in iPS derived cardiomyocytes when compared to isogenic controls.

### Sample Protocol
Samples were processed in an Orbitrap Fusion Tribid using the TMT-MS3 method

### Data Protocol
Data was processed using Proteome Discoverer software and Perseus.
