### Title
Protein correlation of Mus musculus tissues

### Description
SILAC based protein correlation profiling using size exclusion of protein complexes derived from Mus musculus tissues (Heart, Liver, Lung, Kidney, Skeletal Muscle, Thymus)

### Sample Protocol
Samples were processed using in solution digestion followed by C18 STAGE tip clean up.

### Data Protocol
DATA was processed using MAXQUANT against the Mus musculus uniprot database
