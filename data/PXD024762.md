### Title
Breast cancer cells treated with milk extracellular vesicles

### Description
4T1 breast cancer cell treated with bovine milk extracellular vesicles

### Sample Protocol
Cell lysates were subjected to Gel-LC-MS/MS analysis.

### Data Protocol
Peak lists were extracted from raw mass spectrometry into the Mascot Generic File Format (MGF) using MsConvert with peak picking. The MGF files were then searched using X! Tandem.
