### Title
DOM composition changes in a warming peatland

### Description
Measure changes in dissolved organic matter composition and resulting microbial decomposition rates in an experimentally warmed peatland.

### Sample Protocol
Samples were digested with trypsin, then analyzed by LC-MS/MS

### Data Protocol
Data was searched with MS-GF+ using PNNL's DMS Processing pipeline.
