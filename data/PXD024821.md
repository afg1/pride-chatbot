### Title
Milk extracellular vesicles administered to mice

### Description
Liver of mouse treated with milk extracellular vesicles

### Sample Protocol
Liver tissue was homogenised and subjected to Gel-LC-MS/MS

### Data Protocol
Peak lists were extracted from raw mass spectrometry into the Mascot Generic File Format (MGF) using MsConvert with peak picking. The MGF files were then searched using X! Tandem.
