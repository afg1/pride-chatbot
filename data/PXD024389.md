### Title
Proteomics of Zinc-limited Mycobacterium tuberculosis

### Description
Mycobacterium tuberculosis H37Rv (Mtb) grown in chemically defined medium replete for Zinc and limited for Zinc

### Sample Protocol
Peptides were prepared by FASP. Proteins were treated with iodoacetamide and digested on FASP columns using Trypsin. Peptides were purified, desalted and dried. Peptides were sent to the UC Davis proteomics core for LC-MS/MS.

### Data Protocol
.RAW files were converted into .mzXML format using RawConverter. Data searches were performed using X! Tandem via GPM Fury 3.1.
