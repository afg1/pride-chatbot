### Title
A human gut microbiome treated in vitro with Diclofenac

### Description
One individual microbiome was cultured with or without the presence of diclofenac (an NSAID drug) in triplicates using RapidAIM assay.

### Sample Protocol
The protein digests were analyzed using a 1.5-hour gradient with Orbitrap Q-Exactive.

### Data Protocol
MetaLab Desktop (V2.2) was used to search the six samples against the IGC database using default settings of closed search.
