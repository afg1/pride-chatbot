The structure of each dataset used to create the dataset DB is the following: 

PXD{}.md This is the name of each file. Which contains: 

- Title
- Description
- Data protocol
- Sample protocol
- Instrument
- From the SDRF the Unique sample information is key-value pair; for example: disease: []; organism:[]; etc.
- publication abstract: Get the publication accession (PubMed ID) get the abstract of the publication and include it here.
